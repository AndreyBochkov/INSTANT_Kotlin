# INSTANT_Kotlin

Мессенджер, заточенный на приватность и быстродействие.
Вы смотрите на код Android-приложения.

### Структура проекта

```
.
├── data/
│   ├── Entity.kt ................................... Типы данных
│   └── PageType.kt ................................. Типы страниц
├── MainActivity.kt ................................. Точка входа
├── network/
│   ├── INSTANTWS.kt ................................ Логика взаимодействия с сервером
│   └── INSTANTWSMessage.kt ......................... Типы сообщений между клиентом и сервером
├── security/
│   ├── AESKeyStoreHelper.kt ........................ Помощник хранения ключей в Android KeyStore
│   ├── EncryptedStorage.kt ......................... Помощник безопасного хранения данных
│   ├── HandshakeHelper.kt .......................... Помощник в алгоритмах рукопожатия с сервером
│   └── INSTAESHelper.kt ............................ Помощник с AEC-GCM
├── ui/
│   ├── composable/
│   │   ├── INSTANT*Page.kt ......................... Страницы приложения
│   │   ├── INSTANTApp.kt ........................... Точка входа композиции
│   │   ├── INSTANTTopAppBar.kt ..................... Верхняя полоска
│   │   └── INSTANTViewModel.kt ..................... Слой взаимодействия UI и внутренней логики
│   └── theme/ ...................................... Сгенерировано MaterialBuilder
└── utils/ .......................................... Спецфункции и значения, которые удобно было вынести
```
